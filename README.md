# Fantasy Stock Market Game

A stock trading game powered by Claude AI where you manage a portfolio of fictional companies over 10 trading days. Available with both a modern GUI interface and classic terminal gameplay.

## Features

- **75 Unique Companies**: Claude generates 75 diverse companies across 18 different sectors
- **Modern GUI Interface**: Desktop application with intuitive controls and real-time updates
- **Advanced Filtering**: Search by company name/symbol and filter by sector for easy navigation
- **Dynamic Content Generation**: Claude Sonnet 4 generates unique companies, news events, and market analysis for each playthrough
- **18 Industry Sectors**: Tech, Energy, Food, Entertainment, Healthcare, Defense, Finance, Retail, Automotive, Aerospace, Real Estate, Biotech, Consumer Goods, Industrial, Telecom, Utilities, Materials, Transportation
- **Realistic Market Simulation**: Stock prices move based on news, volatility, and market sentiment
- **4-Phase Trading Rounds**: Market News, Trading Window, Price Resolution, and Market Recap
- **Interactive Trading**: Click-to-buy/sell interface with real-time portfolio updates
- **Live Charts**: Portfolio performance visualization with matplotlib integration
- **Special Events**: Merger rumors, market crashes, meme stock frenzies every 3 rounds
- **Rich Terminal UI**: Color-coded price movements and clean portfolio display (also available)
- **Performance Grading**: Get graded S-D based on your investment returns
- **Save/Load Functionality**: Save your game progress and resume later

## Setup

1. **Install Dependencies**:
```bash
pip install -r requirements.txt
```

2. **Set up API Key** (Optional - for full Claude AI experience):
   - Get API key from https://console.anthropic.com
   - Copy `.env.example` to `.env`
   - Add your Anthropic API key: `ANTHROPIC_API_KEY=your_key_here`
   - Without API key, game runs in test mode with 75 predefined companies

3. **Run the Game**:

**Easy Launcher (Recommended):**
```bash
python start_game.py
```

**Manual Launch:**
```bash
# GUI Version
python gui_app.py

# Terminal Version  
python main.py
```

## Game Structure

### Company Generation:
- **75 unique companies** generated by Claude at game start
- **18 diverse sectors** with 3-5 companies per sector
- Each company has unique backstory, volatility, and stability

### Each Trading Day (10 rounds):
1. **Market News**: 3 headlines (global, sector, company-specific)
2. **Trading Window**: Buy/sell stocks with unlimited trades
3. **Price Resolution**: Claude calculates new prices based on news
4. **Market Recap**: Dramatic CNBC-style analysis of the day

### Special Events (every 3 rounds):
- Merger Rumors
- Market Crashes  
- Meme Stock Frenzies
- Bubble Warnings
- Insider Tips

## Grading System

- **S Tier**: +100% return or more
- **A Tier**: +50% to +99%
- **B Tier**: +20% to +49%
- **C Tier**: +0% to +19%
- **D Tier**: Negative returns
